一个简单的 web 版 JVM 监控器

作者： 风筝 / 2023-06-02

 ·  本文被阅读次

Java, JVM

以 JMX 技术为基础，差不多实现了一个 web 版的 visualVM，具备以下功能：

支持连接本地 JVM 和 远程 JVM 展示所有 MBeans，并可查看属性及调用操作 仪表盘功能： 实时系统信息，系统 CPU 使用率、内存占用等 JVM 参数信息 实时 CPU 使用率曲线图、实时 Heap 、MetaSpace 使用量曲线图、实时类加载量曲线图、实时线程数曲线图 垃圾收集器及垃圾收集次数
